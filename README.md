# MiraiBot
一个供用户使用的基于Mirai-Api-Http的Mirai插件。
# 安装
1.从 [Release](https://github.com/xiaodi001-01/MiraiBot/releases) 获取我的项目文件
2.下载你所需要的版本
3.解压文件，把启动脚本也移动到项目根目录
4.运行
